# 微博每日一句鸡汤运营AI员工 PRD

## 1. 项目概述

### 1.1 项目背景
基于现有的150条优质鸡汤文案库（`sentence.md`），通过AI自动配图，并自动发布到微博，降低人工运营成本，提高内容发布的稳定性和持续性。

### 1.2 项目目标
- 实现每日自动化内容生成和发布
- 保证内容质量通过人工审核机制
- 提升微博账号活跃度和用户粘性
- 充分利用现有文案资源，减少内容创作成本

### 1.3 目标用户
- 微博运营团队
- 内容审核人员

### 1.4 现有资源
- **文案库**：`D:\university\Pudow\onesentence-oneday\sentence.md`
- **文案数量**：150条优质鸡汤文案
- **文案特点**：积极向上、适合早晨发布、有共鸣感

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 文案选择模块
**功能描述**：从现有文案库中选择文案

**输入**：
- 文案库路径：`D:\university\Pudow\onesentence-oneday\sentence.md`
- 选择策略：顺序/随机/智能推荐
- 选择数量：30条

**输出**：
- 30条文案及其序号（1-150）
- 避免短期内重复

**选择策略**：
- **顺序模式**：按文案库顺序依次选择（默认）
- **随机模式**：随机选择未使用的文案
- **智能模式**：根据历史互动数据优先选择高互动文案

**去重机制**：
- 记录已使用文案的ID和使用时间
- 确保30天内不重复
- 当文案库接近用完时提前预警（剩余<10条）

#### 2.1.2 AI配图生成模块
**功能描述**：根据文案内容生成匹配的图片

**输入**：
- 文案内容
- 图片风格偏好（清新/温暖/简约等）
- 图片尺寸（建议1080x1080或16:9）

**输出**：
- 与文案匹配的AI生成图片
- 图片格式：PNG（用于后续添加logo）
- 图片质量：适合社交媒体发布

**质量要求**：
- 图片与文案主题相关
- 视觉效果美观，色彩和谐
- 无敏感内容
- 符合微博图片规范

#### 2.1.3 Logo水印添加模块
**功能描述**：在生成的图片上添加企业logo水印

**输入**：
- AI生成的背景图片
- Logo文件路径：`D:\university\Pudow\onesentence-oneday\logo\`
- Logo选择策略（根据背景色自动选择）

**Logo资源**：
- **原色版本**：`PUDOW朴道健康水专家-原色.png`（浅色背景使用）
- **反白版本**：`PUDOW朴道健康水专家-反白.png`（深色背景使用）
- **墨稿版本**：`PUDOW朴道健康水专家-墨稿.png`（备用）

**水印配置**：
- **位置**：右下角
- **大小**：背景宽度的12-15%
- **边距**：距离右边和底边各20px
- **透明度**：85-90%（保证清晰但不抢眼）

**智能选择逻辑**：
```python
# 根据背景图右下角区域的平均亮度自动选择logo版本
if 背景亮度 > 128:
    使用原色logo  # 深色logo适合浅色背景
else:
    使用反白logo  # 白色logo适合深色背景
```

**输出**：
- 带企业logo的最终图片
- 图片格式：JPG（压缩后发布）

#### 2.1.4 内容审核模块
**功能描述**：人工审核生成的内容

**审核界面需求**：
- 展示30组文案+配图
- 支持单条编辑/删除/替换
- 支持批量通过/拒绝
- 标记问题内容（敏感词、质量不佳等）

**审核流程**：
1. 系统生成30组内容
2. 通知审核人员
3. 审核人员逐条检查
4. 修改或替换不合格内容
5. 确认通过后进入发布队列

#### 2.1.5 定时发布模块
**功能描述**：每天早上自动发布一条内容到微博

**配置项**：
- 发布时间（默认：每天早上8:00）
- 发布账号
- 发布频率（每天1条）

**发布流程**：
1. 从审核通过的内容池中按顺序取出一条
2. 调用微博API发布
3. 记录发布状态（成功/失败）
4. 发送发布结果通知

**异常处理**：
- 发布失败自动重试（最多3次）
- 内容池不足时提前预警
- API限流时延迟发布

### 6.2 Logo水印处理技术
**实现方案**：使用Python Pillow库进行图像处理

**核心代码示例**：
```python
from PIL import Image
import numpy as np

def add_logo_watermark(bg_path, output_path):
    """在背景图上添加logo水印"""
    bg = Image.open(bg_path).convert("RGB")
    
    # 检测右下角区域亮度，自动选择logo版本
    region = bg.crop((bg.width-200, bg.height-200, bg.width, bg.height))
    brightness = np.array(region).mean()
    
    # 根据背景亮度选择logo
    if brightness > 128:
        logo_path = "logo/PUDOW朴道健康水专家-原色.png"
    else:
        logo_path = "logo/PUDOW朴道健康水专家-反白.png"
    
    logo = Image.open(logo_path).convert("RGBA")
    
    # 调整logo大小为背景宽度的13%
    logo_width = int(bg.width * 0.13)
    logo_height = int(logo.height * logo_width / logo.width)
    logo = logo.resize((logo_width, logo_height), Image.Resampling.LANCZOS)
    
    # 设置位置（右下角，边距20px）
    position = (bg.width - logo_width - 20, bg.height - logo_height - 20)
    
    # 粘贴logo（保留透明度）
    bg.paste(logo, position, logo)
    bg.save(output_path, quality=95)
    
    return "原色" if brightness > 128 else "反白"
```

**技术要点**：
- 自动检测背景亮度，智能选择logo版本
- 保持logo透明度和清晰度
- 固定位置和比例，保证品牌一致性
- 处理速度快（<2秒/张）
