# 🎨 前端图片编辑器使用指南

## 📦 安装依赖

```bash
cd frontend
npm install
```

**主要依赖：**
- Vue 3 - 前端框架
- Vue Router - 路由管理
- Pinia - 状态管理
- Element Plus - UI 组件库
- Fabric.js - Canvas 图形编辑库
- Axios - HTTP 客户端

---

## 🚀 启动开发服务器

### Windows:
```bash
cd frontend
start_dev.bat
```

### macOS/Linux:
```bash
cd frontend
npm run dev
```

默认地址：**http://localhost:5173**

---

## 🎯 功能概览

### 1. **内容列表页** (`/content`)
- 查看所有生成的内容
- 筛选和搜索功能
- 批量审核操作
- 点击**"编辑图片"**按钮进入编辑器

### 2. **图片编辑器** (`/editor/:id`)

#### **核心功能：**

✅ **AI 背景生成**
- 点击"生成背景图"按钮
- 自动调用云雾 AI 生成图片
- 等待 30-60 秒
- 背景自动加载到画布

✅ **文字层编辑**
- 自动添加初始文案
- 可拖拽移动位置
- 调整字号、行高、对齐方式
- 修改字体、颜色
- 支持粗体、斜体
- 多行文本支持

✅ **Logo 水印**
- 点击"添加 Logo"
- 可拖拽到任意位置
- 切换版本：原色/反白/墨稿
- 调整大小和透明度

✅ **图层管理**
- 右侧图层列表
- 点击选中图层
- 查看和修改属性
- 删除不需要的图层

✅ **导出图片**
- 点击"导出图片"
- 自动下载 PNG 格式
- 分辨率：1024x1024

## 🎨 使用流程

### 步骤 1：进入编辑器
1. 访问 http://localhost:5173
2. 进入"审核池"页面
3. 选择一条内容
4. 点击**"编辑图片"**按钮

### 步骤 2：生成背景
1. 点击工具栏的**图片图标**（生成背景图）
2. 等待 AI 生成（30-60秒）
3. 背景自动加载到画布

### 步骤 3：编辑文字
1. 点击画布上的文字
2. 右侧属性面板显示文字属性
3. 修改内容、字号、颜色等
4. 拖拽调整位置

### 步骤 4：添加 Logo
1. 点击工具栏的 **Logo 图标**
2. Logo 出现在画布右下角
3. 拖拽到合适位置
4. 在属性面板切换版本和调整大小

### 步骤 5：导出图片
1. 点击**"导出图片"**按钮
2. 图片自动下载到本地
3. 文件名格式：`pudow-content-{id}-{timestamp}.png`

## 🖱️ 快捷操作

### Canvas 操作：
- **点击** - 选中图层
- **拖拽** - 移动图层位置
- **拖拽角** - 调整大小（按住 Shift 等比缩放）
- **拖拽旋转** - 旋转图层（角落圆圈）
- **Delete 键** - 删除选中图层

### 工具栏按钮：
- 🖼️ **生成背景图** - 调用 AI 生成背景
- **T** **添加文字** - 添加新文字层
- 📦 **添加 Logo** - 添加 Logo 水印
- 🗑️ **删除选中** - 删除当前选中的图层
- 💾 **导出图片** - 下载最终图片

## 📝 注意事项

### ⚠️ Logo 文件位置
确保 Logo 文件放在 `frontend/public/` 目录：
```
frontend/public/
  ├── PUDOW朴道水汇-横-原色.png
  ├── PUDOW朴道水汇-横-反白.png
  └── PUDOW朴道水汇-横-墨稿.png
```

### ⚠️ API 配置
编辑 `frontend/.env` 或 `frontend/.env.development`：
```env
VITE_API_BASE=https://weibo-daily-sentence.zeabur.app
```

本地开发时可以改为：
```env
VITE_API_BASE=http://localhost:8000
```

### ⚠️ CORS 跨域
确保后端允许前端域名访问（已在后端配置）

## 🐛 常见问题

### Q: 生成背景图一直转圈？
**A:** 检查：
1. 后端服务是否运行正常
2. Zeabur 环境变量是否配置（OPENAI_API_KEY 等）
3. 浏览器控制台是否有错误信息

### Q: Logo 不显示？
**A:** 检查：
1. Logo 文件是否在 `frontend/public/` 目录
2. 文件名是否正确
3. 浏览器控制台是否有 404 错误

### Q: 文字无法编辑？
**A:** 
1. 确保图层已选中（有蓝色边框）
2. 在右侧属性面板编辑
3. 或双击文字直接编辑

### Q: 导出的图片不完整？
**A:**
1. 确保所有图层都在画布内
2. 检查浏览器是否阻止下载
3. 尝试刷新页面重新导出
