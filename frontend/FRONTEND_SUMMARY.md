### 1. 项目架构搭建

## 🚀 如何启动

### Windows 用户

```bash
# 1. 安装依赖
cd frontend
setup.bat

# 2. 启动开发服务器
start_dev.bat

# 3. 访问
浏览器打开: http://localhost:3000
```

### Linux/Mac 用户

```bash
# 1. 安装依赖
cd frontend
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问
浏览器打开: http://localhost:3000
```

## 🔗 API 对接

前端通过 Vite 代理连接后端：

```javascript
// vite.config.js
proxy: {
  '/api': {
    target: 'http://localhost:8000',
    changeOrigin: true
  }
}
```

所有 API 请求自动转发到后端服务。

## ⚠️ 注意事项

### 1. 开发环境
- 需要 Node.js 16+ 和 npm 7+
- 后端服务必须在 8000 端口运行
- 建议使用 Chrome 或 Edge 浏览器

### 2. 生产部署
- 需要配置 Nginx 代理或后端 CORS
- 修改图片 URL 处理逻辑（改为实际域名）
- 构建命令：`npm run build`

### 3. 兼容性
- 支持现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）
- 不支持 IE11
